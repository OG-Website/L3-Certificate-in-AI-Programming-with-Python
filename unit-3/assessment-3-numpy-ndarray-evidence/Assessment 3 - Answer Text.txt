Q1. What NumPy is used for and what an ndarray is

NumPy is a third-party Python library used for numerical computing. It is useful when working with arrays, matrices and large groups of numbers. The main NumPy data structure is the ndarray, which means n-dimensional array. An ndarray can be one-dimensional, two-dimensional or higher-dimensional, and it stores data in a structured numerical form.

NumPy is used because it makes mathematical and data-processing operations easier than using normal Python lists. For example, an ndarray can store a table of scores, measurements or dataset values, and NumPy can calculate across the whole array efficiently. NumPy arrays also have useful properties such as shape, ndim, size and dtype. These properties show the structure, number of dimensions, number of elements and data type of the array.

Q2. How to create, save and load NumPy ndarrays

An ndarray can be created after importing NumPy with import numpy as np. A common method is np.array(), which creates an ndarray from existing data such as a list or nested list. In the notebook, og_scores = np.array([[82, 76, 91], [68, 88, 95]], dtype=int) creates a two-dimensional ndarray with two rows and three columns. The dtype=int argument stores the values as integers.

An ndarray can be saved with np.savetxt or np.save. np.savetxt saves a readable text or CSV-style file, while np.save saves NumPy's native .npy format. The .npy format is useful because it preserves the ndarray structure and data type for later loading. The notebook proves the process by loading the saved files back with np.loadtxt and np.load.
