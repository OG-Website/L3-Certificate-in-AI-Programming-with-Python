Statistical Analysis with Pandas and Titanic CSV

Q1. Explain how Pandas can be used to perform statistical analysis on a dataset.

Pandas can be used to perform statistical analysis by loading structured data into a DataFrame and then applying inspection, summary and calculation commands. A DataFrame is suitable because it stores data in rows and columns, which makes it practical for analysing CSV files such as titanic.csv. In the Unit 3 statistical analysis work, the Titanic CSV file is loaded with pd.read_csv and stored as titanic_df. This creates a table-like object that can be inspected and analysed with Pandas commands.

The first step is to check that the data has loaded correctly. Commands such as titanic_df.head(), titanic_df.shape and titanic_df.dtypes help confirm the first rows, the number of rows and columns, and the data type of each column. This matters because statistical analysis should not be carried out blindly. If the wrong file is loaded, or if a numeric column has been read as text, the analysis could be misleading.

Pandas can then calculate summary statistics. The command titanic_df.describe() gives useful information for numeric columns, including count, mean, standard deviation, minimum, quartiles and maximum. Separate commands can also be used for individual measures, such as mean(), median(), mode() and std(). These measures help explain the dataset. For example, mean and median can describe central tendency, while standard deviation describes spread.

Pandas can also group data before calculating statistics. For example, titanic_df.groupby("Sex")["Survived"].mean() compares survival rates by sex, and titanic_df.groupby("Pclass")["Survived"].mean() compares survival rates by passenger class. This is stronger than only calculating one overall average because it shows patterns between categories.

The advantage of using Pandas is that it gives short, readable commands for analysis that would take much longer with plain Python. A limitation is that the results depend on the quality of the data. Missing values, incorrect data types or misunderstood categories can affect the analysis. For evidence, the notebook should show the code, the output and a short explanation of what each output proves.

Q3. Provide annotated code evidence for the statistical analysis.

The following annotated code shows the practical statistical analysis used for the Unit 3 Titanic dataset. The comments explain what each part of the code does and why it is useful as evidence.

from pathlib import Path
import pandas as pd

# Locate the Titanic CSV file in the current folder.
# If it is not there, check a data subfolder as a backup.
csv_path = Path("titanic.csv")
if not csv_path.exists():
    csv_path = Path("data") / "titanic.csv"

# Load the CSV file into a Pandas DataFrame.
# A DataFrame is suitable because the Titanic data is arranged in rows and columns.
titanic_df = pd.read_csv(csv_path)

# Show the number of rows and columns to prove that the dataset loaded.
print("Rows and columns:", titanic_df.shape)

# Show the first five rows so the assessor can see the actual dataset structure.
print("First five rows:")
print(titanic_df.head())

# Check the data types before analysis.
# This helps confirm which columns are numeric and suitable for statistics.
print("Data types:")
print(titanic_df.dtypes)

# Produce summary statistics for the numeric columns.
# describe() gives count, mean, standard deviation, minimum, quartiles and maximum.
print("Summary statistics:")
print(titanic_df.describe())

# Select numeric columns for individual statistical measures.
numeric_columns = ["Age", "Fare", "SibSp", "Parch", "Survived"]

print("Mean values:")
print(titanic_df[numeric_columns].mean(numeric_only=True))

print("Median values:")
print(titanic_df[numeric_columns].median(numeric_only=True))

print("Mode values:")
print(titanic_df[numeric_columns].mode().head(1))

print("Standard deviation values:")
print(titanic_df[numeric_columns].std(numeric_only=True))

# Group by category to compare survival rates.
print("Survival rate by sex:")
print(titanic_df.groupby("Sex")["Survived"].mean())

print("Survival rate by passenger class:")
print(titanic_df.groupby("Pclass")["Survived"].mean())

# Check missing values to include data quality inspection.
print("Missing values by column:")
print(titanic_df.isna().sum())
